Welcome

Complete guide to managing and customizing your Cake Decorating Studio website.

Congratulations on your new website! Sweet Frosting is a high-performance, premium web template designed specifically for boutique bakeries and custom cake decorators. This documentation will walk you through everything from setup to advanced customization.

The website is built with modern web technologies, ensuring it is 100% responsive, SEO-friendly, and lightweight.

Installation Guide

How to set up your website for local development and live hosting.

Local Setup

To view the website on your computer, follow these steps:

  1. Extract Files: Unzip the project folder to your desired location.
  2. Editor: Open the folder using a code editor like VS Code.
  3. Live Server: Right-click on index.html and select "Open with Live Server".
# Example: Opening in VS Code code .

Hosting & Deployment

To make your website live, we recommend the following platforms:

  • Vercel (Recommended): Drag and drop the folder into the Vercel dashboard.
  • Netlify: Use the "Deploy manually" feature by uploading the folder.
  • Github Pages: Push the code to a repository and enable Pages in settings.

Customization

Easily adapt the website to match your unique brand identity.

Colors (CSS Variables)

All brand colors are stored as CSS variables in style.css. Modify these to change the theme site-wide.

:root { --color-primary: #FF5D8F; /* Primary Rose Color */ --color-secondary: #2C3333; /* Deep Charcoal Text */ --color-accent: #D4AF37; /* Gold Highlights */ }

Fonts

The site uses Google Fonts. To change them, replace the links in the <head> of your HTML files.

  • Headings: Times New Roman / Playfair Display
  • Body Text: Outfit (Sans-serif)

Content Replacement

To update the text or images, locate the following markers in the HTML files:

  • Hero Section: Look for the <h1> tag near the top.
  • Images: Update the src="..." attribute in <img> tags.
  • Contact Info: Search for the "Contact" section footer to update your email and phone.

Pro Tip: Use "Find and Replace" (Ctrl+H) to update your brand name across all pages at once.
